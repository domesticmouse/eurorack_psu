# Eurorack PSU

A simple 6HP +-12V Eurorack PSU, based on Winterbloom's [Micronova][].
See [EurorRack dimensions][] for faceplate dimensions.

[Micronova]: https://winterbloom.com/shop/micronova/
[EurorRack dimensions]: https://midisoft.de/EuroRackDimensions/EuroRack_Dimensions.html
